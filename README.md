# LANTools
> 学习 golang 练手小项目

![](doc/image.png)

## Usage
Accepter type "accept" to accept file, sender type "send" and choose a file to send

接收方输入 accept 进入待接收状态，发送方输入 send

需要帮助可以输入 help 获取指令信息
